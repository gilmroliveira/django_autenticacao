# 📝 Django Auth System - Termux Edition

Este projeto consiste em um sistema de autenticação completo (Cadastro, Login e Dashboard) desenvolvido inteiramente em um ambiente mobile utilizando o **Termux**.

---

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.11+
* **Framework:** Django 5.0+
* **Ambiente:** Termux (Android)
* **Versionamento:** Git & GitHub
* **Segurança:** Autenticação via Chave SSH (ED25519)

---

## 🚀 Como o projeto foi elaborado

1. **Preparação do Ambiente:** Configuração do Python e Django dentro do Termux.
2. **Desenvolvimento MVT:** Criação de rotas, views e templates para Login e Cadastro.
3. **Versionamento:** Uso de branches para isolar o desenvolvimento da autenticação.
4. **Segurança:** Implementação de Chave SSH para superar as restrições de senha do GitHub no mobile.
